# Reproducible Clinical Data Analysis – Cardiovascular Disease & AMI

Reproducible R workflow for cardiovascular disease and AMI analysis 
using real-world clinical data. Covers data auditing, missing data 
handling, leakage prevention, and exploratory analysis following 
biomedical data science best practices.

## Objectives
1. Identify baseline variables associated with cardiovascular disease.
2. Evaluate prognostic factors in patients with acute myocardial infarction.

## Methods
- **Data audit:** structure inspection and consistency checks against data dictionary
- **Missing data:** pattern evaluation, complete-case analysis, and simple imputation
- **Plausibility checks:** clinical range validation for numeric variables
- **Leakage prevention:** two temporally separated analytical datasets
- **Exploratory analysis:** distribution, outcome comparison, descriptive statistics
